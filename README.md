# Astrit-Gentiana
Ftesë per Darsem
Create a highly premium, ultra-luxury digital wedding invitation website for Astrit & Gentiana. The design should be royal, elegant, and modern with black and gold colors, cinematic animations, glowing effects, and smooth transitions.

Include:

* Animated royal monogram “A G” as a logo
* Opening cinematic screen (click to enter experience)
* Background particle effects (gold luxury style)
* Wedding details: 25 July at 18:00, Restaurant California, Milloshevë, Kosovo
* Google Maps location button
* RSVP form (name + attending / not attending buttons)
* The RSVP responses should be sent via Formspree (email notification system)
* Background romantic instrumental music (starts after user interaction)
* Countdown timer to the wedding date
* Fully responsive mobile design
* Glassmorphism UI (luxury transparent card style)
* Smooth animations and hover effects

The result should feel like a high-end VIP wedding invitation website with a cinematic and emotional experience.”**
